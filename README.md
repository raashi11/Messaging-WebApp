# messaging-webapp
Project Setup:
1. git clone https://github.com/raashi11/messaging-webapp.git
2. pip install -r requirements.txt
3. python manage.py runserver
4. open http://localhost:8000/admin/

Admin login credentials:-
User: Branch_International
password: branch

password: <user_id>
